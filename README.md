一套自己DIY的脚手架，支持构建React、Vue2、Vue3项目（Vue2、Vue3尚未导入模板，目前仅支持React项目构建）。

**🚧 施工中**

## 安装

```shell
npm i tust-cli -g
# cnpm i tust-cli -g
```

## 使用

```shell
tust-cli -v # 查看版本
tust-cli create # 创建项目
```
